How It Works (Split Architecture)

BomboFlipper offloads heavy calculations to a background Node.js service, guaranteeing zero client lag and peak FPS in Minecraft.

Node.js Service

Backend Engine

  • 1.
    Bazaar Sync: Fetches real-time commodity prices from the Hypixel Bazaar API every 60 seconds.
  • 2.
    AH Scan & Indexing: Pulls active BIN auctions and establishes a baseline Lowest BIN (LBIN) for clean items.
  • 3.
    NBT Upgrade Valuation: Extracts Recombs, HPBs, Fuming Books, and Enchants, applying live Bazaar depreciation values.
  • 4.
    Volume & Demand Analysis: Calculates daily sales volume to assign items a Demand Tier (1–5).
  • 5.
    WebSocket Broadcast: Beams verified flips over local WebSocket (port 8080).
Fabric 1.21.11 Mod

Client Mod

  • 1.
    WebSocket Connection: Connects directly to the local backend port 8080 with auto-reconnect.
  • 2.
    Client Filter Pipeline: Enforces user rules for Budget, Min/Max Profit, Min Demand Tier, and Keyword Blacklist.
  • 3.
    Bounded LRU Deduplication: Prevents duplicate alerts for the same auction UUID.
  • 4.
    Click-to-Buy Chat Alerts: Formats profitable flips into interactive chat messages with sound cues.
  • 5.
    In-Game GUI Menu: Seamless in-game configuration menu & live command sync.

Key Features

Zero Frame Drops

Zero Client Overhead

All HTTP fetching, GZIP decompression, and JSON parsing happen inside the background Node.js service. Your Minecraft client stays at 100% peak FPS.

True Resale Value

Smart Bazaar-AH Hybrid Math

Bounded LRU Cache

Installation & Setup

The backend auction server is already running for you. Simply install the Fabric 1.21.11 mod below to start flipping instantly.

1

Install Minecraft Mod (Fabric 1.21.11)

Zero backend configuration needed

  1. Install Fabric Loader for Minecraft 1.21.11.
  2. Place the bomboflip.jar file into your mods directory:
    .minecraft/mods/bomboflip.jar
  3. Launch Minecraft 1.21.11 (Fabric) and log in to Hypixel Skyblock.
  4. The mod automatically connects to the pre-configured backend service and streams instant auction flip alerts to chat.
Type /bomboflipper in chat anytime to open in-game settings.

Command Reference

Instant in-game command controls. Adjust your filters without leaving your auction house workflow.

/bomboflippercore

Opens the visual configuration menu in Minecraft.

/bomboflipper enable <true/false>core

Enables or disables the auction sniper engine.

Example: /bomboflipper enable true
/bomboflipper chatAlerts <true/false>core

Toggle interactive chat notification displays.

Example: /bomboflipper chatAlerts true
/bomboflipper soundAlerts <true/false>core

Toggle sound cues when a profitable flip is detected.

Example: /bomboflipper soundAlerts true
/bomboflipper budget <amount>financial

Set maximum purchase budget limit in coins.

Example: /bomboflipper budget 50000000
/bomboflipper minProfit <amount>financial

Set minimum projected profit threshold in coins.

Example: /bomboflipper minProfit 1000000
/bomboflipper maxProfit <amount>financial

Set maximum profit cap to filter pricing bait (0 for unlimited).

Example: /bomboflipper maxProfit 100000000
/bomboflipper minDemandTier <1-5>financial

Only beam alerts meeting or exceeding this demand tier rating.

Example: /bomboflipper minDemandTier 3
/bomboflipper blacklist add <item>blacklist

Blacklist items containing specific keywords.

Example: /bomboflipper blacklist add Rune
/bomboflipper blacklist remove <item>blacklist

Remove a keyword from your active blacklist.

Example: /bomboflipper blacklist remove Rune
/bomboflipper blacklist clearblacklist

Clear all blacklisted keywords from configuration.

Example: /bomboflipper blacklist clear
/bomboflipper debugMode <true/false>debug

Toggle live backend logs, auction cycle runtimes, and demand stats (Tier + Sales/Day) in chat.

Example: /bomboflipper debugMode true
